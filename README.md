# Vending Machine

To Create a Finite State Machine for a Vending Machine Controller that takes Currency and Dispenses Items.

## Installation

MacOS
```
Navigate to project directory using Terminal.

```
Windows
```
Install MinGW
Navigate to project directory using PowerShell.
Type "MinGW32-make"
Type "./Vending.exe"

```
